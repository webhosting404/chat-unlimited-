Cara Menggunakan:

  1. Klik tombol "Hubungkan ke Perangkat Bluetooth" untuk memulai koneksi
  2. Pilih perangkat Bluetooth yang ingin Anda hubungkan (kedua perangkat harus sudah dipasangkan)
  3. Setelah terhubung, Anda dapat mulai mengirim pesan
  4. Pada perangkat lain, lakukan langkah yang sama untuk terhubung dan mulai chatting
  5. Pastikan kedua perangkat mendukung Web Bluetooth API (Chrome/Edge di desktop dan Android)

Catatan: Aplikasi ini menggunakan karakteristik Bluetooth khusus untuk pertukaran pesan. Pastikan kedua perangkat menggunakan aplikasi yang sama.
